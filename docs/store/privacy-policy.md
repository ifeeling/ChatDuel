# ChatDuel 隐私政策

更新日期：2026-06-16

ChatDuel 是一个 Chrome / Edge 浏览器扩展，用来在一个页面里对比 ChatGPT、Gemini、豆包等官方网页的回答。ChatDuel 不提供 AI 服务，不要求 API Key，也不会把用户的聊天内容上传到 ChatDuel 服务器。

## 1. 我们不会收集的内容

ChatDuel 服务器不会收集、保存或分析以下内容：

- 用户输入的问题、提示词或附件内容
- ChatGPT、Gemini、豆包等 AI 平台返回的回答正文
- 用户在官方 AI 平台上的账号、Cookie、Token、Session 或其它登录凭据
- 用户访问的具体网页 URL、官方会话链接或浏览历史
- 可直接识别个人身份的信息，例如姓名、邮箱、手机号、精确地址

## 2. 扩展在本地保存的内容

为了提供历史记录、复制、导出、总结、转发和恢复官方会话等功能，ChatDuel 会在用户自己的浏览器本地存储中保存部分使用数据。

这些数据主要包括：

- 用户通过 ChatDuel 发送的问题和实际发送内容
- 用户在扩展中添加的附件信息
- 扩展捕获到的 AI 回复文本
- 本地历史记录、统计信息和用户设置
- 官方网页的会话链接和当时显示的 AI 面板状态
- 远程选择器配置缓存

这些内容保存在浏览器本地存储中，用于用户自己回看和继续使用。ChatDuel 不会把这些本地历史记录上传到 ChatDuel 服务器。

用户可以通过浏览器扩展管理页移除 ChatDuel，或在扩展内删除相关历史记录，从而清除本地数据。

## 3. 与官方 AI 平台的交互

ChatDuel 会在用户已经登录的官方网页中辅助用户输入、发送和读取页面上的回答。用户主动发送给 ChatGPT、Gemini、豆包等第三方平台的内容，会按照对应平台自己的隐私政策和服务条款处理。

ChatDuel 不控制这些第三方平台如何处理数据。用户使用这些平台时，应同时查看并遵守对应平台的隐私政策和用户协议。

## 4. 配置接口请求

为了在官方网页结构变化时尽快修复输入框、发送按钮等页面选择器，ChatDuel 会请求以下配置接口：

`https://chatduel.ifeeling.app/api/extension/config`

该接口只返回扩展运行所需的选择器配置，不返回广告、脚本代码或用户内容。扩展请求该接口时会带上当前扩展版本号，方便判断不同版本的兼容情况。

## 5. 服务器技术日志

ChatDuel 配置服务器只记录配置接口的最小技术日志，用于排查服务是否可用、配置是否生效、不同扩展版本是否正常获取配置。

记录字段包括：

- 请求时间
- 脱敏后的 IP 地址
- User-Agent
- 扩展版本
- 返回的配置版本
- HTTP 状态码

服务器不会在这些日志中记录用户聊天内容、AI 回复、附件内容、网页 URL、官方会话链接或账号凭据。

## 6. 权限用途

ChatDuel 申请的浏览器权限只用于扩展核心功能：

- `storage` / `unlimitedStorage`：保存本地历史、设置、会话状态和配置缓存。
- `tabs`：识别用户是否已经打开 ChatGPT、Gemini、豆包官方网页，并更新扩展页面里的状态。
- `scripting`：在用户已打开的官方网页中注入必要的内容脚本，辅助输入和读取回答。
- `declarativeNetRequest`：仅在用户主动打开 ChatDuel 页面时，按需调整官方网页的嵌入限制，让网页可以显示在扩展页面中。
- `downloads`：导出本地记录或内容时使用。
- `alarms`：定期刷新远程选择器配置。
- `host_permissions`：仅允许访问 ChatGPT、Gemini、豆包官方网页，以及 ChatDuel 配置接口域名。

## 7. 数据共享

ChatDuel 不出售用户数据，不把用户聊天内容共享给广告网络、数据经纪商或其它第三方。

用户主动发送到 ChatGPT、Gemini、豆包等第三方平台的内容，不属于 ChatDuel 服务器收集或共享的数据；这些内容由对应第三方平台处理。

## 8. 数据安全

ChatDuel 使用 Manifest V3 浏览器扩展机制，并尽量减少远程接口和权限范围。配置接口只通过 HTTPS 域名访问，服务端日志只保留排障所需的最小字段。

任何浏览器扩展都无法保证第三方网页永久稳定可用。如果官方平台页面结构变化，ChatDuel 可能需要更新本地代码或远程选择器配置。
